GraphQL API for Wetu (‘Our’ SM Platform)
-----------------------------------------
This is the Graphql version of https://github.com/abu-jeylani/wetu-rest-api

## Requirements

- Node v18.8 or higher
- NPM
- Git

## Technologies Used

- Node.js
- Express.js
- Express-graphql
- Graphql
- Jsonwebtoken
- MongoDB
- Mongoose
- Express-validator
- Multer
